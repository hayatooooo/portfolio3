# 筋トレ回数記録アプリになります。

# 開発環境

Windows10　×　Ubuntu(Windows Subsystem For Linux)　×　Docker Desktop　×　AWS(EC2)

# 以下、機能になります。

・Log in機能(devise gem 使用。名前とパスワードのみの簡易ログインに修正。)  
・筋トレ回数記録  
・回数に応じて、レーダーチャート、コメント、キャラクターが変化
